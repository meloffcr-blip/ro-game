# مشروع رسالة حب لرحمة ❤️

## وصف المشروع
موقع ويب رومانسي تفاعلي يحتوي على رسائل حب وألعاب مخصصة لرحمة.

## هيكل المشروع
```
├── index.html          # الصفحة الرئيسية
├── assets/
│   ├── css/
│   │   └── style.css   # ملف التنسيقات
│   ├── js/
│   │   └── script.js   # ملف JavaScript
│   └── images/
│       └── mohamed.jpg # صورة محمد
└── README.md           # ملف التوثيق
```

## المميزات
- رسائل حب رومانسية
- ألعاب تفاعلية
- تصميم متجاوب
- دعم اللغة العربية

## كيفية التشغيل
1. افتح ملف `index.html` في المتصفح
2. استمتع بالتجربة التفاعلية

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
- Google Fonts (Cairo)
